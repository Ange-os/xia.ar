# Assets - Imágenes

## Logos de abmSistemas

Para agregar los logos de abmSistemas:

1. **Favicon**: `abmsistemas-favicon.png` o `.ico` (logo cuadrado con "S")
2. **Logo pequeño**: `abmsistemas-logo-small.png` (para footer, altura ~20-30px)
3. **Logo completo**: `abmsistemas-logo.png` (logo completo con texto)

### Especificaciones sugeridas:

- **Favicon**: 32x32px o 64x64px (formato PNG o ICO)
- **Logo pequeño**: Altura 20-30px, mantener proporción
- **Logo completo**: Ancho máximo 200px, mantener proporción
